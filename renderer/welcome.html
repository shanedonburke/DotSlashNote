Welcome to ./note

To begin, create a notebook using the

button.



Notes are stored in notebooks. To create a note, double-click to enter a notebook, then use the

button.

  • Items in the sidebar can be selected by clicking on them. By holding Ctrl or Shift while selecting, multiple items can be selected at once. From there, right click on the selected items to see options.

  • Both notebooks and notes can be stored in folders for easy organization. Notebooks can also be exported as a PDF file by right clicking them and selecting Export.

  • Notes are saved automatically upon switching tabs. The open note will save automatically every minute. To save the open note manually, use Ctrl+S or go to File - Save

  • ./note supports code highlighting through highlight.js. In the note editor, create a code block and begin typing. ./note will detect the language, and your code will be highlighted automatically.

  • ./note also supports LaTeX equation formatting through KaTeX. Try it out in the note editor by pressing the Formula button, entering a LaTeX-formatted equation, and pressing Save.

  • A number of open-source frameworks and libraries made the development of ./note possible. To learn more about ./note, go to Help - About.
